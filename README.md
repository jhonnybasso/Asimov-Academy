Projetos — IA no Desenvolvimento Frontend
Dois projetos desenvolvidos com uso intensivo de IA como acelerador de desenvolvimento.

Parte 1 — Positivus Landing Page
Implementação estática da Positivus Landing Page em HTML/CSS puro, baseada no design original disponível na Figma Community.
Stack: HTML5 / CSS3 / JavaScript vanilla
Parte 2 — Hero Section: Curso Python + IA
Hero section para landing page de curso, com fundo animado via Canvas API, partículas conectadas, floating code chips e hierarquia visual completa (badge → headline → bullets → CTAs → social proof).
Stack: HTML5 / CSS3 / JavaScript vanilla · Canvas API · Google Fonts (Inter + JetBrains Mono)

Ferramentas de IA utilizadas

Claude (Anthropic) — geração do HTML/CSS/JS, estrutura de seções, componentes interativos e animações
Gemini (Google) — interpretação de referências visuais e validação de decisões de layout

Etapas em que a IA contribuiu
A IA foi responsável pela maior parte do desenvolvimento em ambos os projetos: conversão do Figma em código (Parte 1), criação do sistema de partículas, gradientes e hierarquia visual (Parte 2), componentes interativos (accordion, carrossel, formulário com abas) e responsividade geral.
Ajustes manuais
Correções pontuais de espaçamento, proporções de elementos e alinhamentos — na Parte 1 para fidelidade ao design original, e na Parte 2 para equilibrar o visual sem comprometer o foco no CTA principal.
